# design 폴더 공통 작성 규칙

- design 폴더 안에 생성하는 문서는 기본적으로 index.html 구조 규칙을 따른다.
- 페이지는 헤더, 본문, 푸터 3영역으로 나눈다.
- 헤더 좌측 메뉴는 홈 링크로 구성한다.
- 헤더 우측 메뉴는 비교, 커뮤니티, 프로필 3개 링크를 사용하고 href는 #으로 둔다.


- 본문에는 그리드 형식 카드 섹션등으로 구성된다. 



- 푸터 배경은 연한 회색, 텍스트는 검정으로 사용한다.
- 푸터 메뉴는 회사소개, 알림, 로그인 링크를 넣고 href는 #으로 둔다.
- 구현 언어는 html, css, javascript만 사용한다.

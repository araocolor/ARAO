### 3. 이미지 압축 (Sharp)
- 관리자 이미지 업로드 시 자동 압축
- **thumb**: 800px / WebP quality 80 (랜딩 페이지용)
- **full**: 1600px / WebP quality 87 (갤러리 상세용)
- 원본 2.2MB → thumb ~150KB / full ~450KB
- `beforeImage` / `afterImage` = thumb URL
- `beforeImageFull` / `afterImageFull` = full URL
- Supabase Storage `landing-assets` 버킷에 저장

### 4. YouTube 영상 섹션
- 랜딩 홈 리뷰 다음에 유튜브 영상 섹션 추가
- 썸네일 클릭 시 재생 (hover 시 재생 버튼 빨간색)
- `LandingContent.video` 구조로 관리자 페이지에서 수정 가능
- YouTube URL / youtu.be 단축 URL 모두 파싱 지원

### 5. 갤러리 페이지 개편
- 히어로 섹션 3개로 확장: **인물 / 환경야외 / 실내카페**
- Before 버튼 누르는 동안만 before 사진 표시, 떼면 after 복귀
- 모바일에서 이미지 좌우 여백 없이 풀 너비 표시
- full 버전 이미지 사용으로 고화질 유지

### 6. 기타

- `next.config.ts`에 `allowedDevOrigins` 추가 (로컬 IP 테스트용)
